# Calculadora de Notas | UTEC

Esta es una calculadora de notas para estudiantes de UTEC. Permite ingresar las calificaciones de cinco evaluaciones y calcular la nota final.

## Funcionamiento

- Ingresa las calificaciones de las cinco evaluaciones.
- Haz clic en el botón "Calcular" para obtener la nota final.
- Se mostrará la tabla con las calificaciones ingresadas y la nota final calculada.

## Instrucciones de uso

1. Abre el archivo `index.html` en tu navegador web.
2. Ingresa las calificaciones en los campos correspondientes.
3. Haz clic en el botón "Calcular".

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Bootstrap (framework de CSS)

## Estructura de archivos

- `index.html`: Contiene la estructura HTML del proyecto.
- `css/estilos.css`: Archivo CSS con los estilos personalizados.
- `js/calcularNotaFinal.js`: Archivo JavaScript que realiza los cálculos de las notas.
